# Weather-Based-Activity-Suggestion-App
This project suggests suitable activities based on the current weather conditions of a given city.
The user enters a city name, the app fetches real-time weather data, then recommends an activity such as jogging, reading indoors, traveling, etc., depending on the weather.

Features:

User enters a city name

Fetches live weather data (temperature, humidity, conditions)

Suggests activities according to weather

Simple and user-friendly UI

Tech Stack:

Java

Spring Boot

REST API Integration

HTML, CSS, JavaScript
